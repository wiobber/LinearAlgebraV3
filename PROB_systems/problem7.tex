\begin{exercise}
Find all solutions (if any) to the system.  

$$\begin{matrix}
      x& +&2y&-&z&=&2\\
      2x & +&5y&-&3z&= &1\\
      x&+&4y&-&3z&=&3
    \end{matrix}$$

 $$(\answer{DNE}, \answer{DNE}, \answer{DNE})$$
If the system is inconsistent, type DNE in each answer cell.
\end{exercise}
